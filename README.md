# MacGamePort

Play Windows games on your Mac. Download the latest notarized build below — drag it to
Applications and open. No account needed.

### Download

**→ [Latest release](https://github.com/seanellul/macgameport-dist/releases/latest)**

The app is signed with a Developer ID and notarized by Apple, so it opens cleanly with no
Gatekeeper warning. It updates itself automatically (via Sparkle) when new builds ship.

_Requires macOS 14 (Sonoma) or later on Apple Silicon._
